## Installation

**Conan:**
```sh
pip install conan
```

**Linux:**

```sh
chmod 777 Linux
./Linux
```

**Windows:**

```sh
.\windows.bat
```

## Usage Example

**In Terminal**

```sh
./r-type_server
```

**In another terminal**

```sh
./r-type_client
```

## Controller

**Player**

 Button        | Direction
 --------------|-------------
 Arrow Up      | Top
 Arrow Down    | Down
 Arrow Left    | Left
 Arrow Right   | Right
 Space         | Shoot
